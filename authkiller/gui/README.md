# THINKER GUI 使用说明

## 简介

THINKER GUI 是 AuthKiller 的桌面图形界面应用，基于 Tkinter 开发，提供直观的密码测试操作界面。

## 功能特性

### 1. 主窗口布局
- 配置表单：多标签页配置界面
- 进度显示：实时进度条和统计信息
- 结果表格：成功凭证列表展示
- 日志显示：实时运行日志

### 2. 协议选择
- HTTP 表单登录（POST/GET）
- HTTP Basic 认证

### 3. 字典配置
- 用户名字典文件选择
- 密码字典文件选择
- 字典预览功能
- 三种攻击模式：
  - 普通模式（所有用户名×所有密码）
  - 单用户模式（一个用户名×所有密码）
  - 密码喷洒模式（一个密码×所有用户名）

### 4. 并发和超时设置
- 并发数设置（1-100）
- 请求超时设置（1-60秒）
- 重试次数设置
- 断点续传配置

### 5. 任务控制
- 启动任务
- 暂停/恢复任务
- 停止任务
- 实时进度监控

### 6. 实时进度更新
- 进度条显示
- 统计信息：
  - 总尝试次数
  - 成功次数
  - 失败次数
  - 剩余组合数
  - 成功率
  - 耗时
  - 平均速度

### 7. 结果展示和导出
- 成功凭证表格展示
- 支持搜索和过滤
- 导出格式：
  - JSON
  - CSV
  - TXT
- 右键菜单操作：
  - 复制用户名/密码
  - 查看详情
  - 删除结果

### 8. 日志显示
- 实时日志输出
- 日志级别过滤（DEBUG/INFO/WARNING/ERROR）
- 日志颜色标记
- 清空和导出功能

### 9. 预留功能
- 语言选择接口（支持简体中文、英语等多语言扩展）
- 主题定制接口

## 安装依赖

```bash
pip install aiohttp colorama pyyaml
```

## 启动方法

### 方法1：直接运行
```bash
python -m authkiller.gui.run_gui
```

### 方法2：导入运行
```python
from authkiller.gui import ThinkerApp

app = ThinkerApp()
app.run()
```

## 使用流程

1. **配置目标**
   - 选择协议类型
   - 输入目标URL
   - 设置请求方法和请求体模板
   - 配置自定义Headers

2. **配置字典**
   - 选择用户名字典文件
   - 选择密码字典文件
   - 选择攻击模式
   - 预览字典内容

3. **设置性能参数**
   - 设置并发数
   - 设置超时时间
   - 配置断点续传

4. **配置检测规则**
   - 设置成功状态码
   - 配置成功/失败响应模式
   - 设置防御检测选项

5. **启动任务**
   - 点击"启动"按钮
   - 观察实时进度和日志
   - 查看成功凭证

6. **导出结果**
   - 选择导出格式
   - 保存结果文件

## 文件结构

```
authkiller/gui/
├── __init__.py          # GUI模块初始化
├── thinker_app.py       # 主应用入口
├── main_window.py       # 主窗口布局
├── config_panel.py      # 配置面板
├── progress_panel.py    # 进度面板
├── result_panel.py      # 结果面板
└── run_gui.py           # 启动脚本
```

## 核心集成

GUI应用已完全集成以下核心模块：

- **ConfigManager**: 配置管理
- **AttackEngine**: 攻击引擎
- **DictionaryManager**: 字典管理
- **Reporter**: 结果报告生成

## 注意事项

1. 请确保字典文件存在且可读
2. 合理设置并发数，避免对目标造成过大压力
3. 遵守法律法规，仅用于合法的安全测试场景
4. 使用前请先配置好所有必要参数
5. 运行过程中可以暂停或停止任务
6. 结果会自动保存到 results.json 文件
